# MarketMind

## ML-Powered Product Launch Feedback Simulator

MarketMind is an AI/ML-powered product validation platform that helps users test product ideas before launch. It analyzes real customer review data using NLP, sentiment analysis, complaint mining, and customer clustering, then generates real-time feedback from ML-backed virtual customer personas.

## Current Phase

Phase 0: Project setup and repository structure.

## Planned Features

- Product review data analysis
- Sentiment analysis
- Complaint mining
- Customer persona clustering
- Product launch scoring
- Live AI Customer Panel
- Streamlit dashboard
